== Description ==

Easy way to view and manage all images.

1. All the images are now arranged by one table.
2. This plugin is useful for find unused images.
3. You can easily rename all the files.

   
= Features =
1. List all images in the media library
2. Export to CSV
3. Edit Filename
4. Edit ALT
5. View Filesize
6. View if the image is used or not (published in posts or pages)

== Installation ==

1. Copy the entire /images-list/ directory into your /wp-content/plugins/ directory.
2. Activate the plugin.
3. New Tab called Images List will be genereate.


== Frequently Asked Questions ==
Can I rename image that is already in used in some posts? Yes, the plugin is searching the posts that's related to the image and change the url.
What about the thumbnails? The plugin is regenerating the thumbnails based on the new filename
